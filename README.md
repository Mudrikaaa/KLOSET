## ✨ Overview

**KLOSET** is a cross-platform mobile app (iOS, Android & Web) that acts as an intelligent personal stylist. It helps users make the most of the clothes they already own — with outfit recommendations tailored to each person's **body type**, **skin tone**, and **style preference**.

> Democratizing personal styling with an accessible, AI-powered stylist.

---

## 🚀 Highlights

- 🧥 **Wardrobe Management** — Add and organize clothing by category, all in one place.
- 💫 **Outfit Discovery** — A swipe-based interface for browsing personalized outfit suggestions.
- 🎯 **Occasion-Based Styling** — Curated looks for events like interviews, weddings, and casual outings.
- 👤 **Personalized Profiles** — Recommendations that adapt to your body type, skin tone, and style.
- 📱 **Cross-Platform** — One experience across iOS, Android, and Web.
